# Ask Tool Reference

Queries the SwiftZilla Swift/iOS knowledge base (Cloud) for best practices and architecture tradeoff guidance.

## Execution Syntax

```bash
./swiftzilla/scripts/swiftzilla ask "<NATURAL_LANGUAGE_QUESTION>"
```

## Constraints & Requirements

*   **API Key**: Mandatory `SWIFTZILLA_API_KEY`.
*   **Purpose**: Use this ONLY for general Swift/iOS questions. For project-specific questions, use `context query`.

## Output Handling

*   Analyze and present the technical guidance directly to the user.
